## 📖 Overview
The **Shuttle Management System** is a full‑stack web application designed to manage shuttle operations efficiently.  
It provides administrators with tools to handle routes, schedules, and passenger information, while offering a user‑friendly interface for daily operations.

---

## ✨ Features
- 🚌 **Shuttle Scheduling** – Create and manage shuttle timetables  
- 📍 **Route Management** – Define and update shuttle routes  
- 👥 **Passenger Management** – Register and track passengers  
- 📊 **Admin Dashboard** – Monitor operations and generate reports

## 🚀 Getting Started

### Prerequisites
- Java 17  
- Maven 3.9+  
- MySQL 8.0+  

### Installation
```bash
# Clone the repository
git clone https://github.com/souhayl66/shuttle-management.git

# Navigate into the project
cd shuttle-management

# Build the project
mvn clean install

# Run the application
mvn spring-boot:run
